# 1. HTML5 规则校验

> 用法：在 input 标签中

| 关键字   | 用法                   | 作用                      |
| :------- | :--------------------- | :------------------------ |
| required |                        | 必填项                    |
| pattern  | pattern="1\d{10}"      | 输入需满足正则表达式      |
| min/max  | type="number" max="42" | 输入的最大值和最小值      |
| step     | step="2"               | 输入的步长                |
|          | type="url"             | 输入需是有效的 URL        |
|          | type="email"           | 输入需是有效的 Email 地址 |
